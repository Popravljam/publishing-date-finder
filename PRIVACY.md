# Privacy Policy for Publishing Date Finder

**Last Updated:** November 24, 2025

## Overview

Publishing Date Finder is committed to protecting your privacy. This extension is designed to work entirely within your browser and does not collect, store, or transmit any personal information.

## Data Collection

**We do NOT collect any data.** Specifically:

- ✅ No personal information is collected
- ✅ No browsing history is tracked or stored
- ✅ No user identifiers or analytics are collected
- ✅ No cookies or tracking mechanisms are used
- ✅ No data is sent to external servers (except as noted below)

## Permissions

The extension requires the following permissions:

### `activeTab`
- **Purpose:** Access the currently active web page to analyze its content for publication dates
- **Data Access:** Only when you explicitly click the extension icon
- **Data Storage:** All analysis happens locally in your browser; no data is stored

### `https://archive.org/*`
- **Purpose:** Query the Wayback Machine API when you click "Check Archive"
- **Data Sent:** Only the URL of the current page (when you explicitly request it)
- **Data Received:** Archive availability information from Archive.org
- **Third Party:** Archive.org has its own privacy policy

## How It Works

1. When you click the extension icon, it analyzes the current page's HTML for date metadata
2. All date detection happens **locally in your browser**
3. If you click "Check Archive," the current page's URL is sent to Archive.org's public API
4. No data is stored between sessions
5. No data is transmitted to any server controlled by us

## Third-Party Services

The only external service used is:

### Wayback Machine (Archive.org)
- **When:** Only when you explicitly click "Check Archive"
- **What's sent:** The URL of the current page
- **Purpose:** To find archived snapshots of the page
- **Privacy Policy:** https://archive.org/about/terms.php

## Data Storage

- ✅ No data is stored persistently
- ✅ No local storage, cookies, or IndexedDB is used
- ✅ All processing is done in memory and discarded when you close the popup
